# 🚀 Business Name Generator

A modern Business Name Generator built using HTML, CSS, and JavaScript. The application generates unique startup and brand names using random word combinations and provides an interactive user experience.

---

## ✨ Features

* Generate unlimited business names
* Interactive and responsive UI
* Copy generated names with one click
* Randomized startup name combinations
* Clean and modern design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
